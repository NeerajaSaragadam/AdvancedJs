Var:
var declaration are globlly scoped or function/locally scoped
when we declare var in outside the function, we can access any where in the window
when we declare var in inside the function, we can access only in the function,we cannot access outside the function
var variable are updated and redeclare in the scope.


Let:

variable declared in a block with let, is only available with in that block, we cannot access outside of the block
let is block scoped.
let can be updated and redeclare

Const:

variable declared with the const maintain constant variable. we cant modify the value of the variable.
const is block scoped.
but we can update the object and array elemenens..we cannot redeclare.
const variable neither be redeclare and updated
